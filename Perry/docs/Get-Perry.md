# Get-Perry

## SYNOPSIS
Get latest errors.

## SYNTAX

```
Get-Perry [<CommonParameters>]
```

## DESCRIPTION
Get latest errors logged by Perry, error-message with script stack and exceptions are available.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-Perry
```

Get a list of latest errors.
