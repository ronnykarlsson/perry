# Clear-PerryStackTraceFilter

## SYNOPSIS
Clear all stack trace filters.

## SYNTAX

```
Clear-PerryStackTraceFilter [<CommonParameters>]
```

## DESCRIPTION
Clear all stack trace filters.

## EXAMPLES

### Example 1
```powershell
PS C:\> Clear-PerryStackTraceFilter
```

Clears all stack trace filters.
